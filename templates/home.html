FWI Prediction Model

Our model predicts the Forest Fire Weather Index (FWI) based on environmental parameters.

Model Type: Linear Regression with Exhaustive Feature Selection (EFS)
Use Case: Early warning system for forest fire management, risk assessment, and prevention.

How it Works:

  • Collect environmental data (Temperature, RH, Wind, Rain, etc.)
  • Preprocess the data (cleaning & scaling)
  • Apply Exhaustive Feature Selection to select best features
  • Train Linear Regression model on selected features
  • Predict FWI for new data inputs
